🦾
Manipulation Robotics
조작 로봇공학
  • Grasping and Dexterous Manipulation
    물체를 잡고 정교하게 조작하는 기술 (예: 계란을 깨뜨리지 않고 잡기, 나사 돌리기)
  • Object Recognition and Pose Estimation
    물체가 무엇인지 인식하고 3D 공간에서의 위치와 방향을 파악하는 기술
  • Multi-modal Sensing
    시각, 촉각, 힘 센서를 동시에 활용해 물체를 인식하고 조작하는 기술
📍 주요 응용 분야
산업 조립라인, 협동 로봇(cobot), 원격 수술
🚗
Mobile Robotics
이동 로봇공학
  • Autonomous Navigation and SLAM
    로봇이 스스로 길을 찾아가며 동시에 지도를 만드는 기술 (자율주행의 핵심)
  • Multi-robot Systems and Swarm Robotics
    여러 로봇이 협력해 작업하는 시스템 (드론 편대비행, 창고 로봇들)
  • Outdoor and Field Robotics
    실외 환경에서 작동하는 로봇 (농업용, 탐사용, 구조용 로봇)
📍 주요 응용 분야
자율주행차, 무인 배송로봇, 실내 자율주행, 드론 군집 비행, 물류창고 다수 로봇 운용
🤝
Human-Robot Interaction
인간-로봇 상호작용
  • Social Robotics and Companion Robots
    사람과 자연스럽게 상호작용하는 로봇 (페퍼 로봇, 반려 로봇)
  • Collaborative Robotics (Cobots)
    사람과 함께 일하는 협업 로봇 (공장에서 사람 옆에서 함께 작업)
  • Natural Language Interface for Robots
    음성이나 자연어로 로봇에게 명령을 내리는 기술
📍 주요 응용 분야
사회적 로봇, 재활·치료 로봇, 협동 작업
🏥
Medical Robotics
의료 로봇공학
  • Surgical Robotics
    수술을 보조하거나 직접 수행하는 로봇 (다빈치 수술 로봇)
  • Rehabilitation and Assistive Robotics
    재활치료를 돕거나 장애인을 보조하는 로봇
  • Bio-inspired and Soft Robotics
    생체 모방 및 부드러운 소재로 만든 로봇 (문어 다리 같은 로봇)
📍 주요 응용 분야
정밀 수술, 재활 치료, 고령자 돌봄
👁️
Perception and Sensing
인지 및 감지
  • Computer Vision for Robotics
    로봇이 카메라로 주변을 이해하는 기술 (물체 인식, 장애물 감지)
  • Multi-modal Sensor Fusion
    여러 종류의 센서 정보를 합쳐서 더 정확한 판단을 내리는 기술
  • 3D Scene Understanding
    3차원 공간을 이해하고 분석하는 기술 (깊이 인식, 공간 구조 파악)
📍 주요 응용 분야
SLAM(지도 작성), 객체 인식, 사람 동작 인식
🧠
Learning and Control
학습 및 제어
  • Robot Learning and Imitation Learning
    로봇이 스스로 학습하거나 사람을 따라하며 배우는 기술
  • Reinforcement Learning for Robotics
    시행착오를 통해 최적의 행동을 학습하는 기술
  • Optimal Control and Motion Planning
    가장 효율적인 경로와 동작을 계획하고 제어하는 기술
📍 주요 응용 분야
로봇팔의 물체 집기, 보행 로봇 균형 제어
🌐
Foundation Models for Robotics
파운데이션 모델 기반 로보틱스
  • Vision-Language-Action (VLA) 모델
    로봇의 행동을 언어로 취급하여 강력한 VLM을 로봇 제어에 미세조정
  • 웹 스케일 지식 전이
    인터넷 규모 데이터로 학습된 모델의 일반화된 세계 지식을 물리적 로봇에 이전
  • 의미론적 추론
    훈련 데이터에 명시적으로 포함되지 않은 추상적 명령 이해 및 수행
📍 주요 응용 분야
RT-2 (Google), RoboCat (DeepMind), 자기 개선 파운데이션 에이전트
✋
Contact-Rich Manipulation
접촉-집중 조작
  • 힘/토크 센서 기반 제어
    물체와의 접촉 상황에서 정밀한 힘 제어를 통한 조작
  • 다단계 장기 과제(Long-horizon Tasks)
    서랍 열기 → 물건 꺼내기 → 테이블에 놓기 등 순차적 작업
  • Dynamic Manipulation
    던지기, 잡기 등 동적인 조작 기술
📍 주요 응용 분야
정밀 조립, 요리 로봇, 일상생활 보조
🔄
Sim-to-Real Transfer
시뮬레이션-실제 전이
  • Domain Randomization
    시뮬레이션 매개변수를 무작위화하여 실제 환경 차이에 강건하게 훈련
  • System Identification
    실제 환경의 동역학 모델을 학습하고 온라인 정책 조정
  • 생성 모델 기반 데이터 증강
    GAN, VAE, 확산 모델을 통한 고품질 합성 훈련 데이터 생성
📍 주요 응용 분야
시뮬레이션 훈련 → 실제 로봇 배포
🚶
Humanoid Robotics
휴머노이드 로봇공학
  • Bipedal Locomotion
    두 다리로 걷는 기술 (동적 균형 제어, 보행 패턴 생성)
  • Whole-body Control
    전신을 협조적으로 제어하는 기술
  • Human-like Interaction
    인간과 유사한 형태와 동작을 통한 상호작용
📍 주요 응용 분야
안내 로봇, 서비스 로봇, 재난 구조
